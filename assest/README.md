- **Format:** JPG or WebP (for better compression)

## Technical Notes

### Optimization:
- Compress images for web (use TinyPNG, ImageOptim, or similar)
- Convert to WebP format for better performance
- Implement lazy loading (already included in JavaScript)
- Use appropriate alt texts for accessibility

### File Naming:
- Use descriptive, hyphenated names
- Keep consistent naming convention
- Include dimensions in filename if multiple sizes

### Backup Images:
Consider having backup images in case primary images don't load or for different screen sizes.
